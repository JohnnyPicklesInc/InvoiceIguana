🦎 InvoiceIguana

A free invoice maker where the whole invoice lives inside the link. Type it in, share the URL — no signup, no watermark, and nothing is ever uploaded or stored on a server. Need a receipt instead? That's built in too.

File format reference

JSON — required: seller, items (each name + price, qty optional). Optional: selleraddress, sellercontact, buyer, buyeraddress, buyercontact, invoicenumber, issuedate, duedate, currency, subtotal, discount, tax, taxlabel, total, paymentinstructions, notes.

{
  "seller": "Acme Consulting",
  "buyer": "Client Co",
  "invoicenumber": "INV-2026-001",
  "items": [ { "name": "Consulting hours", "qty": 10, "price": 150 } ],
  "tax": 80
}

CSV — metadata rows key,value; items item,name,qty,price.

seller,Acme Consulting
buyer,Client Co
tax,80

      How it works

      Your invoice is compressed and encoded straight into the URL — the link is the invoice. When your client opens it, their browser decodes it locally. No database, no account, no expiry, and we never see your data (the part of a URL after # is never sent to any server).
