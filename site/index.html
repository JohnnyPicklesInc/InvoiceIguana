File format reference

JSON — required: seller, items (each name + price; qty optional; optional per-line discount with discounttype of percent or amount). Optional: selleraddress, sellercontact, buyer, buyeraddress, buyercontact, invoicenumber, issuedate, duedate, currency, subtotal, discount, tax, taxlabel, total, paymentinstructions, notes.

{
  "seller": "Acme Consulting",
  "buyer": "Client Co",
  "invoicenumber": "INV-2026-001",
  "items": [ { "name": "Consulting hours", "qty": 10, "price": 150 } ],
  "tax": 80
}

      An invoice that lives in a link

      Fill in the form, and InvoiceIguana compresses the whole invoice into the URL itself. Share that link by email, text, or chat — when your client opens it, their browser rebuilds the invoice locally. There's nothing to sign up for and nothing to store.

      The link is the invoice

      No database, no account, no expiry. The document is encoded straight into the URL, so the link is all you (or your client) ever need to keep.
