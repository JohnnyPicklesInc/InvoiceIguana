Frequently asked questions

Is InvoiceIguana really free?

Yes — free forever, no signup, no watermark, no per-invoice fee. Everything runs in your browser, so there's nothing to charge for.

Where is my data stored?

In your browser (IndexedDB). Every invoice, client, and business you save lives on this one device. Nothing is uploaded. Use Download backup to grab a JSON copy you can import on another browser.

What happens if I clear my browser data?

Anything you saved to this device gets wiped with the rest of the site data. Grab a backup first if you have anything you want to keep. The backup file is a single JSON file you can import later or on another browser.

How do I turn an invoice into a PDF?

Click print / save as PDF. Your browser's print dialog opens — pick Save as PDF as the destination. The page hides the app chrome so only the invoice ends up in the file.

What is the shareable link for?

The whole invoice is compressed into the URL after the #. Browsers never send that part to a server, so the link is the invoice — email or text it, and your client's browser rebuilds it locally. No signup on their end either.

How is "save to this device" different from the shareable link?

Save = private, on your device, only you can see it. Link = a shareable URL that anyone with it can open. Both work independently, so a workflow like "save a private draft, then share the link when it's final" is fine.

Can I edit an invoice after sharing it?

Every result also gives you an edit link that reopens the form with your data and style. Keep that for tweaks; the client link stays read-only.

Do I need an account?

No. Open the page, fill in the form, save or share.
