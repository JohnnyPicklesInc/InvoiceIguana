🦎 Privacy

This page is short because there is very little to disclose.

Your documents never reach us. The entire invoice or receipt is compressed into the part of the link after the # (the URL fragment). Browsers do not send fragments to servers, so neither the generator nor the viewer page ever transmits your document anywhere. There is no database, no account, and nothing to delete — if you have the link, you have the document; if you discard it, it's gone.

Hosting. The site is served by Cloudflare Pages, which processes standard connection data (like your IP address) to deliver the page, as any host does.

No tracking. No cookies, no ads, no third-party scripts. If we ever add privacy-friendly, cookieless page-view counting (Cloudflare Web Analytics), this page will say so.

One exception: logo images. An invoice or receipt can optionally reference a logo image by https:// URL instead of only an emoji. If it does, viewing that document makes a normal image request to whoever hosts that logo — the same way any web page's images work — which reveals your IP address and roughly when you viewed it to that host. This is the one part of InvoiceIguana that isn't fully private, and it only happens if the document's creator added a logo URL.
