File format reference

JSON — required: merchant, items (each name + price, qty optional). Optional: address, contact, date, reference, currency, subtotal, discount, tax, tip, total, payment, footer.

{
  "merchant": "Rat's Deli",
  "items": [ { "name": "Coffee", "qty": 2, "price": 3.50 } ],
  "tax": 0.78
}

        How it works

        Your receipt is compressed and encoded straight into the URL — the link is the receipt. When someone opens it, their browser decodes it locally. No database, no account, no expiry, and we never see your data (the part of a URL after # is never sent to any server).
